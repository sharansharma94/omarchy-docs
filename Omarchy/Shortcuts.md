# Omarchy Shortcuts

These come from `~/.config/hypr/bindings.conf`.

| Key | Action |
| --- | --- |
| `Super + Return` | Terminal in current working directory |
| `Super + Alt + Return` | Tmux in current working directory |
| `Super + Shift + F` | File manager |
| `Super + B` | Browser |

| `Super + Shift + Alt + B` | Private browser |
| `Super + Shift + M` | Spotify |
| `Super + Shift + Alt + M` | Music TUI |
| `Super + Shift + N` | Editor |
| `Super + Shift + T` | Activity monitor |
| `Super + Shift + D` | Docker TUI |
| `Super + Shift + G` | Signal |
| `Super + Shift + O` | Obsidian |
| `Super + Shift + /` | 1Password |
| `Super + Shift + A` | ChatGPT |
| `Super + Shift + Alt + A` | Grok |
| `Super + Shift + Y` | YouTube |
| `Super + Shift + Alt + G` | WhatsApp |
| `Super + Shift + Ctrl + G` | Google Messages |
| `Super + Shift + X` | X |
| `Super + Shift + Alt + X` | X post |

## Core Omarchy Shortcuts

These are from Omarchy's default Hyprland bindings.

| Key | Action |
| --- | --- |
| `Super + Space` | Launch apps |
| `Super + Alt + Space` | Omarchy menu |
| `Super + Escape` | System menu |
| `Super + K` | Show key bindings |
| `Super + Ctrl + E` | Emoji picker |
| `Super + Ctrl + C` | Capture menu |
| `Super + Ctrl + O` | Toggle menu |
| `Super + Ctrl + H` | Hardware menu |
| `Super + Shift + Space` | Toggle top bar |
| `Super + Ctrl + Space` | Background menu |
| `Super + Shift + Ctrl + Space` | Theme menu |
| `Super + Backspace` | Toggle window transparency |
| `Super + Shift + Backspace` | Toggle window gaps |
| `Super + Ctrl + Backspace` | Toggle single-window square aspect |
| `Print` | Screenshot |
| `Alt + Print` | Screen recording menu |
| `Super + Print` | Color picker |
| `Super + Ctrl + Print` | OCR text extraction |
| `Super + Ctrl + L` | Lock system |

## Window Management

| Key | Action |
| --- | --- |
| `Super + W` | Close active window |
| `Ctrl + Alt + Delete` | Close all windows |
| `Super + J` | Toggle window split |
| `Super + P` | Pseudo window |
| `Super + T` | Toggle floating or tiling |
| `Super + F` | Full screen |
| `Super + Ctrl + F` | Tiled full screen |
| `Super + Alt + F` | Full width |
| `Super + O` | Pop window out, float, and pin |
| `Super + L` | Toggle workspace layout |
| `Super + Arrow` | Focus window in that direction |
| `Super + Shift + Arrow` | Swap window in that direction |
| `Super + 1-9,0` | Switch workspace |
| `Super + Shift + 1-9,0` | Move window to workspace |
| `Super + Tab` | Next workspace |
| `Super + Shift + Tab` | Previous workspace |
| `Super + Ctrl + Tab` | Previous active workspace |
| `Super + S` | Toggle scratchpad |
| `Super + Alt + S` | Move window to scratchpad |
| `Super + Left mouse` | Move window |
| `Super + Right mouse` | Resize window |
